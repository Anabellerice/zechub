## This Week's Education Piece 

There's definitely no doubt that Zcashd serves as the core software for running a Zcash node. It allows users to participate in the Zcash network by validating transactions, maintaining a copy of the blockchain and interacting with other participants in the network. It also enables users to send and receive Zcash transactions securely and privately. You can learn more about how you can use Zcashd via the link below ðŸ‘‡
